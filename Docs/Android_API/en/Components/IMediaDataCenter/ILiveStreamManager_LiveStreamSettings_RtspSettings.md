# class RtspSettings

|  |
| --- |
| ``` class RtspSettings ``` |

|  |  |
| --- | --- |
| *Package:* | dji.v5.manager.datacenter.livestream.settings |

##### Description:

It is used to build and get the RTSP live stream settings. Please refer to the following example to obtain the live broadcast address. RTSP setting parameters are as follows: Username: 123456 Password: 123 Port number: 8554 The ip address of the MSDK App installation device: 192.168.2.13

The live stream address is as follows: rtsp://123456:123@192.168.2.13:8554/streaming/live/1 `Supported since MSDK 5.0.0`
