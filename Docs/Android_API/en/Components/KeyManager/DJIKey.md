##### Description:

This class allow users to access the key, you can set and get the status and function of each module. DJIKeyInfo : It means key. For exmaple, DJICameraKey.KeyEnterPlayback means the interface of playback. productId = 0 means the index of connected product, normally is 0. componentIndex = 0 1 2 3 means the index of gimbal, you can refer to ComponentIndexType.java, normally is 0. subComponentType = SubComponentType means lens, zoom = 0, wide = 1, ir = 2，IGNORE=0xffff-1（means all） subComponentIndex Only battery box uses it, it means the accessories in next level, not uses with cameras.
