# class IWPMZManager

|  |
| --- |
| ``` interface IWPMZManager ``` |

|  |  |
| --- | --- |
| *Package:* | com.dji.wpmzsdk.interfaces |

##### Description:

This class is a waypoint mission editing class, which is used for loading waypoint mission files, editing waypint fields and generating waypoint mission files. For the meaning of the waypoint field, please refer to the code comment and [DJI WPML](https://developer.dji.com/doc/cloud-api-tutorial/cn/api-reference/dji-wpml/overview.md). For the interface call process, please refer to: [Wayline Sample](https://developer.dji.com/doc/mobile-sdk-tutorial/en/tutorials/waypoint.md). `Supported since MSDK 5.6.0`

###### method transKMLtoKMZ

| --- |
| ``` boolean transKMLtoKMZ(String kmlImportPath , String kmzOutPath , HeightMode transParams) ``` |

|  |  |
| --- | --- |
| *Package:* | com.dji.wpmzsdk.interfaces |

##### Description:

Convert KML files to KMZ files. When converting non-DJI-generated KML files to KMZ files, height mode needs to be provided. The supported height modes are WGS84, EGM96, and RELATIVE. The RELATIVE mode is only used for DJI Pilot display and developers can ignore it.

##### Input Parameters:

|  |  |
| --- | --- |
| String kmlImportPath | *The KML file path that needs to be converted.* |
| String kmzOutPath | *KMZ file path stored after conversion.* |
| HeightMode transParams | *Height mode.* |

##### Return:

|  |  |
| --- | --- |
| boolean | *`true` indicates that the conversion was successful.* |
